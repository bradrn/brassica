Brassica is an advanced sound change applier with features including multigraphs and phonetic categories. This page runs Brassica 0.0.3.

You are currently on the online version of Brassica. You may also want to try the standalone desktop interface, which may be downloaded from the GitHub repository. This online version does not support all features of Brassica: in particular it does not yet include opening or saving, or syntax highlighting. It may also somewhat slow. However, otherwise it should be fully compatible with the desktop version of Brassica.

For more information on how to use Brassica, see the official documentation.
